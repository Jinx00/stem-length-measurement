# HagPF: Hierarchical-annotation-guided Phenotypic Framework for Stem Instance Segmentation and Length Measurement in plant point clouds


## Dataset
This dataset (file: data.zip) (https://doi.org/10.5281/zenodo.19564655) contains 663 sets of colourless point cloud data for four dicotyledonous crops (tobacco, tomato, soya bean and chilli pepper).
